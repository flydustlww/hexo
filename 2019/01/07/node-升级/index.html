node 升级

brew search node

即可看到当前可用的版本

brew install node@6

安装需要版本

brew unlink node

首先取消当前版本

brew link node@6 [–force]

注意：在切换版本的时候，可能会需要用到 –force命令，强制执行。
在切换版本时，可能会有一些文件需要删除，注意提示内容，执行即可

export PATH=/usr/local/python/bin:/usr/local/node/bin:$PATH

source /etc/profile

ln -s /home/kun/mysofltware/node-v0.10.28-linux-x64/bin/node(node路径) /usr/local/bin/node

ln -s /home/kun/mysofltware/node-v0.10.28-linux-x64/bin/npm /usr/local/bin/npm

npm config set prefix /usr/local/
